Список требований:

  • Сайт, созданный на любом бесплатном хостинге
  • Эссе по возможностям Вашего хостинга
  • Модуль поиска от любого поискового сервиса
  • 2 видео в проигрывателях разных стандартов
  • Модуль сбора статистики от любого сайта или топа
  • Cтраница, целиком написанная на HTML
  • Страница с собственным стилем CSS
  • Пример JavaScript
  • BOM
  • DOM
  • jQuery

Описание сайта и выполнения требований:

В качестве хостинга был выбран GitHub - сервис GitHub Pages. Суть заключается в том, что создавая репозиторий на GitHub, можно разместить исходники статического сайта и показывать их как сайт. В данном случае, это полезно так как необходимо разместить простой сайт и не беспокоиться о деталях работы с хостингом плюс весь функционал GitHub для работы с репозиторием.
На портале есть 3 html-страницы: index.html (стартовая), video.html (страница с видео), text.html (эта страница); каждая страница имеет свой CSS-стиль: indexStyle.css, videoStyle.css и opisStyle.css соответственно.
Модуль поиска - Яндекс.Поиск (может не работать, т.к. на момент написания текста сайт еще не был проиндексирован).
Модуль сбора статистики - Яндекс.Метрика.
Два видео - одно в проигрывателе HTML5, другое в проигрывателе YouTube.
Пример JavaScript и BOM - кнопка возврата на главную.
Пример DOM - кнопка "Вверх".
Пример jQuery - кнопка "Вверх" и слайдер изображений на главной.
